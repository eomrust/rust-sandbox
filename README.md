Playing around with rust.

# rustbox (Developer Environment)

 ```
 sh presubmit.sh
 cargo build --release
 ```
 
# Troubleshooting

What version of rust am I running?

```
rustc --version
cargo --version
```

Where is rust installed?

```
/usr/local/cargo/bin/rustc
```
